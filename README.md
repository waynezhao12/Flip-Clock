# 翻页钟壁纸
一个漂亮的翻页中壁纸，运行于 Wallpaper Engine 平台。

Steam 创意工坊链接：https://steamcommunity.com/sharedfiles/filedetails/?id=1324135971

由于觉得很好看很适合做壁纸，于是就改成了时钟。
